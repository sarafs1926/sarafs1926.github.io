Princeton Prelims

In January 2019 I took Princeton's Graduate Preliminary Exam. Here are a few resources I created while studying. Please email me if you have comments!

Prelim problem classifiers for each subject. One useful way to study for the prelims is to solve problems from past prelims. There are many ways to solve each prelim problem, but some ways are much easier than others. When studying, I found it useful to classify these problems and order the problems within each section from 'Easy' to 'Hard'. Several friends found this classifier useful as they studied for the 2020 prelims. I have cropped these documents because I haven't finished LaTeXing my handwritten solutions (many of which are still sitting in a large cardboard box). I do plan on releasing full versions of these documents eventually.
